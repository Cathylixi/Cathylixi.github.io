Scientific Tools & Packages

NESS

NESS is an R package that evaluates and improves the quality of low-dimensional embeddings from single-cell RNA sequencing (scRNA-seq) data. It is compatible with popular embedding methods such as t-SNE, UMAP, and phateR, and provides tools for assessing stability, selecting hyperparameters, and visualizing uncertainty.

Medical Agent

An LLM-based medical report OCR and structured data extraction system for echocardiography reports.

This system implements an intelligent medical report processing workflow with the following core functionalities:

  • 📸OCR Text Recognition - Supports text extraction from JPG images and PDF files
  • 🤖AI-Driven Report Classification - Automatically identifies report types such as echocardiography, coronary CTA, etc.
  • 📊Structured Data Extraction - Converts unstructured text into standardized tabular data
  • 🖥️Visualization Interface - Provides a GUI to display structured results
  • 🌍Flexible Deployment - Supports switching between domestic and international DashScope services
